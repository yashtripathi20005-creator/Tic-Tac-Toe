# Tic-Tac-Toe

A classic Tic-Tac-Toe game with a graphical user interface built using Python and tkinter.

## Features

- Clean and modern GUI
- Two-player gameplay (X and O)
- Win detection with visual feedback
- Draw detection
- New game and quit functionality
- Status bar showing current player's turn

## Requirements

- Python 3.6 or higher
- tkinter (included with standard Python installation)
